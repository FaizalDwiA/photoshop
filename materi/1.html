Rectangular Marquee Tool

  1. klik Rectangular Marquee Tool

  2. Seleksi Object

  3. Kembali ke move tool

  4. Copy / tekan ctrl+j(untuk mengcopy sebuah object yang sebelumnya sudah kita seleksi menggunakan Marquee Tool), akan membentuk sebuah layer baru


  5. kita bisa memberi nama pada layer supaya ketika bekerja dibanyak layer kita bisa memberi nama setiap layernya agar nanti tidak bingung.


    6. cara : klik 2X pada tulisan layernya(nama bebas)
