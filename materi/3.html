Brush Tool

  • membuat object dengan brush
  • extensi = .abr

  2. klik gambar ger > Load Brush

  3. cari file brush

  4. pilih nama brush

  5. Klik append

  6. pilih brush > klik di kertas project

  7. harus bikin layer baru agar bisa dipindah2, kalo tidak tidak bisa dipindah
